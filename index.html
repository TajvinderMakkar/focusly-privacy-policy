🎯 Privacy Policy

Focusly - Task Manager

📅 Last Updated: June 12, 2026

📋 Introduction

Welcome to Focusly ("we," "our," or "us"). We are committed to protecting your privacy and ensuring the security of your personal information. This Privacy Policy explains how we collect, use, store, and protect your data when you use the Focusly mobile application.

By using Focusly, you agree to the terms outlined in this Privacy Policy. If you do not agree, please discontinue use of the app.

1️⃣ Information We Collect

1.1 Personal Information

When you create an account or use Focusly, we may collect:

  • Email Address: Used for account creation, authentication, and communication
  • Name: To personalize your experience
  • Profile Information: Display name and avatar (optional)

1.2 Task and Productivity Data

  • Tasks: Titles, descriptions, due dates, priorities, completion status
  • Daily Tasks: Recurring tasks and completion streaks
  • Weekly Planner: Scheduled activities and time blocks
  • Calendar Events: Date-based task organization
  • Analytics Data: Task completion rates, productivity metrics

1.3 Device Information

  • Device Type: Android version, device model
  • App Usage Data: Features accessed, crash reports, performance metrics
  • Notifications: Push notification tokens for reminders

2️⃣ How We Use Your Information

We use your information to:

  • ✅ Provide Core Features: Task management, reminders, scheduling, analytics
  • 🔐 Account Management: Authentication via Firebase and Google Sign-In
  • ☁️ Cloud Sync: Securely backup and sync your tasks across devices
  • 🔔 Send Notifications: Remind you about upcoming tasks and deadlines
  • 📊 Improve the App: Analyze usage patterns to enhance user experience
  • 🛡️ Security: Detect and prevent fraudulent activity

3️⃣ Data Storage and Security

3.1 Local Storage

  • Tasks stored locally using SQLite database
  • Secure storage using Android Keystore

3.2 Cloud Storage

  • Data backed up to MongoDB Atlas (secure cloud database)
  • Encrypted connections via HTTPS/TLS
  • Industry-standard data centers

3.3 Firebase Services

  • Firebase Authentication: Secure login via email/password and Google Sign-In
  • Firebase Analytics: Anonymous usage statistics
🔒 Security Measures:
• End-to-end encryption for data transmission
• Secure authentication tokens
• Regular security audits
• No third-party access to your personal data

5️⃣ Your Rights and Choices

5.1 Access and Update

  • View, edit, and delete tasks anytime
  • Update profile information in Settings

5.2 Data Export

  • Export tasks as JSON or CSV files
  • Download data via Settings → Backup

5.3 Data Deletion

  • Delete Individual Tasks: Swipe left on any task
  • Clear All Data: Settings → Clear All Data
  • Delete Account: Contact us to request full account deletion

6️⃣ Children's Privacy

Focusly is not intended for children under 13 years of age. We do not knowingly collect personal information from children. If we discover that a child under 13 has provided us with data, we will delete it immediately.

7️⃣ Data Retention

  • Active Accounts: Data retained while account is active
  • Inactive Accounts: Data may be deleted after 12 months of inactivity
  • Deleted Accounts: Permanently deleted within 30 days
  • Backups: Retained for up to 90 days for disaster recovery
